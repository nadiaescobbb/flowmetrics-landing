# FlowMetrics Landing Page

Landing page profesional de **FlowMetrics**, plataforma de analítica en tiempo real para equipos que toman decisiones basadas en datos.

---

## Tecnologías utilizadas

- [React 18] – Librería para interfaces interactivas
- [Next.js 13] – Framework para aplicaciones web rápidas y SEO-friendly
- [TailwindCSS] – Framework de CSS moderno y responsivo
- [TypeScript] – Tipado estático para mayor confiabilidad
- [Vercel] – Despliegue y hosting optimizado

---

## Características del proyecto

- **Secciones server-rendered:**  
  - Hero, Social Proof, Features, How It Works, Pricing, Testimonials, Final CTA, Footer
- **FAQ interactivo:** Accordion dinámico con estado manejado del lado del cliente
- **Diseño responsivo:** Compatible con móviles, tablets y escritorio
- **Componentes reutilizables:** Botones, tarjetas, contenedores con estilo consistente
- **SEO-friendly:** Estructura semántica y contenido crawlable

---

## Estructura del proyecto

```

app/
├─ components/
│  ├─ sections/      # Secciones de la landing
│  └─ ui/            # Componentes reutilizables
├─ globals.css        # Estilos globales con Tailwind
├─ layout.tsx         # Layout principal
└─ page.tsx           # Página de inicio

```

---

## Objetivo del proyecto

1. Demostrar **FlowMetrics** como herramienta de analítica
2. Convertir visitantes en leads mediante **CTAs claros**
3. Ofrecer experiencia rápida y responsiva, maximizando **performance y SEO**
4. Mantener un código **modular y limpio**, fácil de mantener y escalar
